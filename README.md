# ИНН Валидатор с проверкой через Dadata

## Описание

Это веб-приложение на React для:
1. Проверки корректности ИНН (структура и контрольные цифры)
2. Запроса информации о компании/ИП через API Dadata.ru

## Функциональность

- Валидация ИНН по алгоритмам ФНС России:
  - Для юридических лиц (10 цифр)
  - Для физических лиц/ИП (12 цифр)
- Автоматический запрос данных из Dadata после успешной проверки
- Отображение основной информации о компании:
  - Полное наименование
  - Юридический адрес
  - ОГРН, КПП, ОКПО
  - Данные о руководителе
  - Статус организации
- Управление API ключом через панель настроек
- Сохранение API ключа в локальном хранилище браузера

## Технологии

- React 18 с TypeScript
- Vite для сборки
- Tailwind CSS для стилизации
- Lucide React для иконок

## Требования

1. Node.js версии 18 или выше
2. npm или yarn
3. Ключ API от Dadata.ru (бесплатный тариф доступен на [dadata.ru](https://dadata.ru))

## Установка и запуск

### Установка зависимостей

```bash
npm install
```

### Запуск в режиме разработки

```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:3000`

### Сборка для production

```bash
npm run build
```

Собранные файлы будут находиться в папке `build`

## Как использовать

1. Запустите приложение командой `npm run dev`
2. Откройте вкладку **Настройки** в приложении
3. Введите ваш API ключ от Dadata.ru и нажмите **Сохранить**
   - API ключ сохраняется локально в браузере и не передается третьим лицам
   - Получить ключ можно на сайте [dadata.ru](https://dadata.ru)
4. Перейдите на вкладку **Поиск**
5. Введите ИНН в текстовое поле (10 или 12 цифр)
6. Нажмите **Найти**
7. Результаты валидации и данные компании отобразятся на странице

## Особенности реализации

1. **Прямой запрос к API** (без прокси-сервера)
   - Работает благодаря CORS-заголовкам Dadata
   - API ключ хранится в localStorage браузера

2. **Валидация ИНН**
   - Проверка формата (только цифры, правильная длина)
   - Проверка контрольных сумм по алгоритмам ФНС России
   - Валидация выполняется локально перед запросом к API

3. **Ограничения:**
   - Лимит 10,000 запросов/сутки (бесплатный тариф Dadata)
   - Некоторые ИНН могут не возвращать данные
   - API ключ хранится в браузере (не рекомендуется для production-решений)
